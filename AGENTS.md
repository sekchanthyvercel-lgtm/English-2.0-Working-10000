# AI Agent Instructions

- **Preserve Unrelated Sections**: NEVER modify code, UI parts, or content that the user did not explicitly request change. If the user is working on "Part C and D", "Part A and B" MUST remain exactly as they were. This is a "Roof and Walls" principle.
- **Strict Styling Fidelity**: Adhere strictly to the requested colors, padding, and layout. Avoid "professional guessing" which introduces unrequested dark modes, black backgrounds, or layout shifts.
- **No Unrequested Guessing**: Do not add "extra" polish like dark backgrounds or shadow effects unless specifically asked. Stick to the light/slate palette unless the user specifies otherwise.
- **Component Specificity**: When fixing a bug in one component, do not refactor other components unless they are directly causing the bug.
- **Style Selection Matching**: Ensure that generated content (MCQs, instructions) exactly matches the visual style selected in the UI. If "Elegant Gold" is selected, use gold/elegant prompts only.
